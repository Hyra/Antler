# Redux Starter
A boilerplate starter app to kickstart your next front-end project

# Features
- [x] React
- [x]  Redux
- [x] Webpack build tool
- [x] Hot Replacement Module for components and reducers
- [x] Redux Dev Tools
- [x] React Router
- [x] Universal Rendering
- [x] Redux Logger
